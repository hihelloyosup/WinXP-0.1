Como rodar este projeto:

    Pré-requisitos: Ter o Node.js instalado no computador.

    Instalação: Abra o terminal na pasta do projeto e digite:
    npm install

    Iniciar: Após instalar, digite o comando abaixo para abrir o app:
    npm run dev (ou npm start)

How to run this project:

Prerequisites: Node.js must be installed on your computer.

Installation: Open the terminal in the project folder and type:

npm install

Start: After installing, type the command below to open the app:

npm run dev (or npm start)
